Welcome to My Memoji Network!

1 NOV 2020    •    tags:

I've been looking online for a graphical way to visually represent some of the connections I've made in the CS, ITI, and Engineering communities during my first year in university but there doesn't seem to be a simple way to do so. I've created this network graph below using Apple's trademark memoji stickers and the Vis.js javascript library.

There are still a lot of things that I'd like to be working on with this project, for example figuring out the best way to display nodes with text in an uncrowded and non-confusing way. Check out the source code if you're interested in doing something similar here
